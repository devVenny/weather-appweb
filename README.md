## 기능

- 도시명으로 검색하면 날씨와 기온정보를 알려준다
- 영상 15도를 기준으로 배경이미지 변경된다

## 추후 추가할 기능

- [O] 기온에 따라 다른 배경이미지 렌더링
- [ ] 검색 키워드, 검색 후 도시명 한글 패치
- [ ] 각 도시별 시간에 맞추어 시간 알려주기
